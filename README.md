# Run Instructions

Run the backend with
```bash
cd backend
./mvnw package && java -jar target/superprice-0.0.1-SNAPSHOT.jar
```
and frontend with
```bash
cd frontend
npm install
npm start
```
Then visit `localhost:3000` to access the site.
The frontend will make requests to the backend to get data from the database.

# Initial Setup

## Setup your environment 
You will need to have in your system

- Java 17.0 or higher
- Node and npm
- Apache Maven
- IDE or Editor

Other tools will be required to complete the project (e.g., Docker)

## Backend

- Delete any unused services (i.e. backend/movies). They are there only for an initial reference.
- Use [Spring initializr](https://start.spring.io/) to create your (micro)services
- Place any new backend service in its own directory (i.e., backend/<service-name>)
- Confirm you can run your applicaiton (./mvnw package && java -jar target/[microservice]-0.0.1-SNAPSHOT.jar)

## Frontend
- cd into frontend/
- Install dependencies "npm install"
- Run the app with "npm start"
